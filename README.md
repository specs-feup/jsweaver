# Jackdaw
Javascript source-to-source compiler based on the LARA framework.

**This is the lara-2.0-legacy version of the compiler.**

**If you want to compile it, do it against the following commits of the [LARA](https://github.com/specs-feup/lara-framework/tree/lara-2.0-legacy) and [specs-java-libs](https://github.com/specs-feup/specs-java-libs/tree/lara-2.0-legacy) repositories**
